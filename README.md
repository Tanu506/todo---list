# 📝 To-Do List

A simple To-Do List built using HTML, CSS, and JavaScript.

## 🚀 Live Demo
👉 https://tanu506.github.io/todo---list/

## ✨ Features

- ✅ Add tasks
- ✅ Delete tasks
- ✅ Mark tasks as completed
- ✅ Add tasks by pressing Enter
- ✅ Prevent empty tasks

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
